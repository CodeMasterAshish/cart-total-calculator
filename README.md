# 🛒 Cart Total Calculator

A simple JavaScript project that calculates the total cost of items in a shopping cart.

## Features

- Add prices for three items
- Calculate total instantly
- Beginner-friendly JavaScript project
- Clean and responsive UI

## Technologies Used

- HTML5
- CSS3
- JavaScript

## How It Works

1. Enter the prices of the items.
2. Click the **Calculate Total** button.
3. The total cart value is displayed on the screen.
